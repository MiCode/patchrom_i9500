Сиз бошқанинг номидан тизимга кирдингиз. Ушбу ойнада кўриб чиқадиган саҳифалар браузер ёки қидирув журналларингизда сақланмайди ҳамда инкогнито ойнани ёпганингизда, cookie-файллар каби из қолдирмайди. Сиз юклаб олган файл ёки яратган веб-ҳужжат сақланади.

Чунки тизимга бошқанинг номидан кириш бошқа фойдаланувчилар, сервер ёки дастурлар хулқига таъсир қилмайди. Қуйидагилардан эҳтиёт бўлинг:

  • Сиз ҳақингиздаги маълумотларни йиғувчи ва баҳам кўрувчи веб-сайтлар
  • Сиз ташриф буюрган саҳифаларни кўриб турувчи Интернет хизмати таъминотчи ва иш берувчилари
  • Сизнинг клавиатурангизни кузатиб турувчи зарарли дастурлар
  • Махфий агентликлар назорати
  • Сизнинг орқангизда турувчи инсонлар
